# card-community-app 社区卡片应用

## 📌 项目简介
本项目是基于 **PHP7.0 后端 + 微信小程序前端** 开发的社区卡片应用，主要用于社区信息展示、用户互动、卡片管理、内容发布等功能。
当前项目为原始工程版本，需要进行**架构升级、功能优化、代码规范、安全性增强、可维护性提升**。

## 🛠️ 现有技术栈
### 后端
- 语言：PHP 7.0
- 服务：原生 PHP / 简易 MVC 结构
- 数据库：MySQL
- 运行环境：Apache / Nginx

### 前端
- 微信小程序原生开发
- WXML + WXSS + JS
- 小程序官方组件与 API

## 📂 项目结构
├── api/ # PHP 后端接口目录
├── config/ # 项目配置文件（数据库、密钥等）
├── pages/ # 微信小程序页面
├── components/ # 小程序公共组件
├── utils/ # 工具函数
├── app.js/app.json # 小程序入口配置
├── static/ # 图片、样式等静态资源
└── README.md
